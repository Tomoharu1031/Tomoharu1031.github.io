Ubuntu20.04でDHCPサーバーを構築したときのメモ

本当のネットワーク初心者なので、お勉強としてメモを残します。

DHCPサーバーとは

IPアドレスを教えてくれるサーバー。 ブロードキャストで探すので、基本的には同じネットワーク内でしか分からない。

IPアドレスについて

  • 192.168.11.0: ネットワークアドレス（自分のネットワーク自身を指すアドレス。ホスト部分が全て0になっているアドレス。）。うちのeno1のアドレス。 192.168.11.1: デフォルトゲートウェイのアドレス。 192.168.11.11: PC（サーバー）のアドレス 192.168.11.255: ブロードキャストアドレス（サブネット（自分のネットワーク）内の全てのホストに一斉送信を行うためのアドレス。）

デフォルトルート：ルーティングテーブル中に宛先IPアドレスに該当するエントリがない場合に使用する経路。 つまり、どこへ送っていいか分からないパケットの送り先

isr-dhcp-serverの設定

isr-dhcp-serverをインストールして、/etc/dhcpd/dhcpd.confを編集。 

    
        option domain-name "xxxxxxx.local"; # 10行目：ドメイン名（DNSサフィックス？）
        option domain-name-servers 192.168.11.1; # 11行目：ネームサーバのホスト名、またはIPアドレス
        authoritative # 24行目
        subnet 192.168.11.0 netmask 255.255.255.0 { # ネットワークアドレスとサブネットマスク
            range 192.168.11.100 192.168.11.120; # IPアドレスを返してくれる範囲。
            option routers 192.168.11.1 # デフォルトゲートウェイ
            option subnet-mask 255.255.255.0 # サブネットマスク
        }
